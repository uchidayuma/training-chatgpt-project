# クレーム対応メール 入力テンプレート

以下の各項目を埋めてから、AIに貼り付けて送信してください。

---

■ 発生した問題：
（例：2024年11月1日に注文した商品が2週間経っても届いていない）


■ お客様の感情・反応：
（例：激怒 / 困惑 / 失望）


■ 対応の方向性：
（例：返金 / 交換 / 謝罪のみ / 現在調査中）


■ 会社名：
（例：株式会社〇〇）


■ 担当者名：
（例：営業部 山田 太郎）

---

**使い方：** このテンプレートの各項目を埋めた後、上記の内容をAIのチャット欄に貼り付けてください。AIが自動的にクレーム対応メールを作成します。
