# 経営向けサマリーの構成基準

## 基本構成（300字以内）

```
①目標対比サマリー（1〜2文）
今週の実績と目標の差分を端的に述べる。

②今週のハイライト（箇条書き2〜4点）
・良い結果：1〜2点
・課題・懸念：1〜2点

③来週の注力点（箇条書き1〜2点）
次に取るべきアクションを具体的に記載する。
```

## 避けるべき表現

| 避けるべき | 代替表現 |
|---|---|
| 「〜の件については引き続き検討中です」 | 「〇〇は△△までに結論を出します」 |
| 「概ね順調に推移しています」 | 「目標比105%で推移、3週連続達成中」 |
| 「色々と取り組んでいます」 | 「〇〇施策を実施し、〜の効果が出ています」 |
| 「〜と思われます」 | 「〜と判断しています（根拠：〜）」 |
| 「なお〜」で始める長い補足 | 別途メール・詳細資料に分ける |

## 読者別の調整ポイント

**経営陣向け：** 数字と示唆のみ。背景説明は最小限。
**部門長向け：** 現場の状況と次のアクションを具体的に。
**チームメンバー向け：** 自分たちの行動と成果の関係が見えるように。
