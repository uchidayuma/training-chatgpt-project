# 引き継ぎ書 抜け漏れチェックリスト（20項目）

## 業務概要
- [ ] 1. 業務名が明確に記載されている
- [ ] 2. 業務の目的・存在理由が書かれている
- [ ] 3. 担当期間（開始〜終了予定）が明記されている

## 作業手順
- [ ] 4. 手順が番号付きで整理されている
- [ ] 5. 各手順に「なぜそうするのか」の背景が1行で補足されている
- [ ] 6. イレギュラー対応（例外処理）が記載されている
- [ ] 7. 手順の中で「初めての人がつまずきやすい箇所」に注記がある

## ツール・システム
- [ ] 8. 使用するツール・システムが列挙されている
- [ ] 9. 各ツールへのアクセス方法（URLやログイン手順）が記載されている
- [ ] 10. アカウント・権限の引き継ぎ方法が明記されている

## 関係者情報
- [ ] 11. 関係者の名前・役割・連絡先が表形式で整理されている
- [ ] 12. 社外関係者（取引先・委託先）も含まれている
- [ ] 13. 「何かあったら誰に確認するか」が明示されている

## 注意事項・トラブル対応
- [ ] 14. よくあるトラブルと対処法が記載されている
- [ ] 15. やってはいけないこと（禁止事項）が明示されている
- [ ] 16. 締め切り・期限が厳しい作業に注記がある

## スケジュール・定期タスク
- [ ] 17. 定期タスクの頻度と実施タイミングが記載されている
- [ ] 18. 月末・四半期末・年度末など特定時期の業務が漏れなく記載されている

## 確認・承認フロー
- [ ] 19. 誰に確認・承認を得る必要があるかが明示されている
- [ ] 20. 成果物・報告先・提出先が明確になっている
