# 業務引き継ぎ情報 入力フォーム

以下の各項目を埋めてからAIに送信してください。引き継ぎ書の構造化された文書を自動生成します。走り書き・箇条書きのままで構いません。

---

■ 業務名・担当期間：
（例：月次請求書処理業務 / 2022年4月〜2024年11月）


■ 業務の目的・背景：
（例：取引先への月次請求書を毎月末に発行し、入金確認まで管理する業務。会計締めに直結するため遅延厳禁）


■ 作業手順（箇条書きや走り書きでOK）：
（できるだけ思い出した順番に書いてください。あとでAIが整理します）


■ 使用ツール・システム：
（例：会計ソフト「freee」、Googleスプレッドシート（管理台帳）、Gmail）


■ 関係者・連絡先（名前と役割）：
（例：田中部長（最終承認者）/ 鈴木さん（経理担当、不明点の相談先）/ 〇〇商事 佐藤様（取引先担当））


■ 注意事項・よくあるトラブルと対処法：
（例：月末28日以降に取引先から請求書の修正依頼が来ることがある→経理の田中部長に相談して差し戻し対応）


■ 定期タスクのスケジュール（あれば）：
（例：毎月25日：請求書下書き作成 / 月末：承認・送付 / 翌月5日：入金確認）

---

**使い方：** 上記を埋めてAIに送信すると、A4 2枚以内の引き継ぎ書（セクション構造付き）が自動生成されます。
