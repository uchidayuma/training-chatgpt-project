# 感情スコア・カテゴリ分類の判定基準

## 感情スコアの判定基準

| スコア | 判定基準 | キーワード例 |
|---|---|---|
| ポジティブ | 満足・感謝・称賛・期待を表す | 「良かった」「助かった」「また使いたい」「おすすめ」「丁寧」 |
| ニュートラル | 事実の報告・要望・提案のみ | 「〜してほしい」「〜があれば良い」「〜だと思う」（感情の強さが弱い） |
| ネガティブ | 不満・失望・怒り・苦情を表す | 「不満」「使いにくい」「改善が必要」「がっかり」「もう使わない」 |

**迷った場合のルール：**
- 否定語（「〜ない」「〜できない」）が含まれる場合はネガティブ優先
- 改善要望だけで感情表現がない場合はニュートラル

## カテゴリ分類の判定基準（デフォルト）

| カテゴリ | 判定基準 |
|---|---|
| 満足度 | サービス・製品全体の評価、総合的な感想 |
| 要望 | 新機能・改善・追加してほしいことへの提案 |
| 不満 | 特定の問題点・不具合・対応への批判 |
| その他 | 上記に当てはまらない、複数カテゴリにまたがる意見 |

**複数カテゴリにまたがる場合：** 最も強い感情が向いているカテゴリを選択する。
